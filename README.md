## Instalation

1. Clone repository: 
```bash
  git clone github.com/davidarizpe/task-list
  cd task-list
```

2. Install dependencies:
```bash
  npm install
```

3. Initialize develompent server:
```bash
  npm run dev
```

4. Initialize preview:
```bash
  npm run preview
```
